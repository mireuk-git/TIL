# Codekata
### 13959
[link](https://www.acmicpc.net/problem/13959)
#### python
```python
d=int(input())
r=int(input())
t=int(input())

r_sum=0
r_curr=4
t_sum=0
t_curr=4-d

while r_sum<r:
    r_sum+=r_curr
    r_curr+=1
    if t_curr>=3: t_sum+=t_curr
    t_curr+=1
    if t_sum>t:
        if t_sum-t == r-r_sum:
            break
print(r-r_sum)
```
- 나이를 기준으로 브루트포스 진행
- `r_sum`,`t_sum`: 각 상자의 현재까지의 초 누적합
- `r_current`,`t_current`: 더할 초 개수
- `t_sum`이 `t`보다 커져야 할 것이기 때문에 정지 조건의 검토는 `t_sum > t`부터 진행, 그 이전엔 빠른 진행을 위해 스킵
- 정지조건은 각각 두 상자의 현재까지의 누적합과 오류값의 차이가 같을 것
