# Codekata
## 16677
[link](https://www.acmicpc.net/problem/16677)
### python
```python
m=input()
n=int(input())
l=[]
for i in range(n):
    w,g=input().split()
    index=0
    for c in w:
        if index==len(m): break
        if c==m[index]: index+=1
    if index==len(m):
        if len(w)==len(m): continue
        g = int(g)/(len(w)-len(m))
        l.append((w,g,i))

if len(l)==0: print("No Jam")
else:
    l.sort(key=lambda x: (-x[1],x[2]))
    print(l[0][0])
```
- `l`: 사전의 글자, 글자 수 대비 갑분싸 정도(`g`), 원래 사전의 순서 저장
- 두 입력의 글자 수 세는 걸로는 조건을 만족하는 지 확인할 수 없다(DEVILIV <-> DVILEVI)
- 원래 입력과 한글자씩 비교해 원본에서 글자를 삽입해 만들 수 있는지 확인하는 걸로 1차 조건 판단, 통과한다면 `l`에 저장
- 모든 단어를 선별해 저장 후 `g`를 기준으로 내림차순, 원래 순서 `i`를 기준으로 오름차순 정렬

### Java
```java
import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;
import java.util.ArrayList;
import java.util.List;
import java.util.StringTokenizer;

class Word{
    String w;
    int index;
    float g;

    Word(String w, int index, float g){
        this.w = w;
        this.index = index;
        this.g = g;
    }
}

public class p16677 {
    public static void main(String[] args)throws IOException{
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        String m = br.readLine();
        int n = Integer.parseInt(br.readLine());
        List<Word> l = new ArrayList<>();
        for (int i=0;i<n;i++){
            StringTokenizer st = new StringTokenizer(br.readLine());
            String w = st.nextToken();
            float g = Float.parseFloat(st.nextToken());
            int idx = 0;
            for (char c: w.toCharArray()){
                if (idx>=m.length()) break;
                if (c==m.charAt(idx)) idx++;
            }
            if (idx>=m.length()){
                if (w.length()==m.length()) continue;
                l.add(new Word(w,idx,g/(w.length()-m.length())));
            }
        }
        if (l.isEmpty()) System.out.println("No Jam");
        else{
            l.sort((a,b) -> {
                if (b.g != a.g) return Double.compare(b.g,a.g);
                return Integer.compare(a.index, b.index);
            });
            System.out.println(l.get(0).w);
        }
    }
}
```
- 단어 `w`, 순서 `index`, 가중치 `g`를 저장하는 `Word` 클래스를 만들어 `List`로 관리, lambda식으로 정렬
