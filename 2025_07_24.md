#### 5705
[link](https://www.acmicpc.net/problem/5705)

```python
d=[0,1,2]
def sequence(n):
    if len(d)>=n+1:
        return d[n]
    d.append(sequence(n-1)+sequence(n-2))
    return d[n]

n=int(input())
while n!=0:
    print(sequence(n))
    n=int(input())
```
- 피보나치마냥 전칸과 전전칸의 이동 가짓수를 더해서 만들어짐
- 재귀함수 ver.

```python
d=[0,1,2]
n=int(input())
while n!=0:
    while len(d)<n+1:
        d.append(d[-2]+d[-1])
    print(d[n])
    n=int(input())
```
- 반복문ver.
- 시간차이는 거의 없고 코드길이만 이쪽이 더 짧다.

```python
import sys
input=sys.stdin.readline

t=int(input())
for _ in range(t):
    n=int(input())
    po5=0
    t=5
    while t<=n:
        po5+=n//t
        t*=5
    print(po5)
```
- 5의 거듭제곱이 팩토리얼의 0 개수에 직접적인 영향을 미친다
- n!이 5의 몇 거듭제곱으로 이루어졌는지 계산
