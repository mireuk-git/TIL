<b>1st try:</b>

```python
import sys
input = sys.stdin.readline

n=int(input().strip())
a,b = map(int,input().strip().split())
points={}
cnt=0
for i in range(n):
    x,y=map(int,input().strip().split())
    if x not in points.keys(): points[x] = set()
    points[x].add(y)

    for dx,dy in [(x+a,y+b),(x+a,y-b),(x-a,y+b),(x-a,y-b)]:
        if dy in points[x]:
            if dx in points.keys():
                if y in points[dx] and dy in points[dx]:
                    cnt+=1
print(cnt)
```
- 딕셔너리를 사용해 입력되는 점들의 x좌표를 기준으로 모아서 저장
- 입력 크기 n<=500000
- 시간초과

- 시간초과 문제 해결

<b>3rd try:</b>

```python
import sys
input = sys.stdin.readline

n = int(input())
a, b = map(int, input().split())

points = set()
for _ in range(n):
    x, y = map(int, input().split())
    points.add((x, y))

cnt = 0
for x, y in points:
    # (x, y)를 왼쪽 아래 모서리로 하는 직사각형이 존재하는지 확인
    if (x + a, y) in points and (x, y + b) in points and (x + a, y + b) in points:
        cnt += 1

print(cnt)
```
- 점의 x,y 좌표를 튜플로 하나로 묶어서 저장
- 어차피 해쉬값으로 저장하는 `set`의 특성 상 검색 범위가 넓어져도 큰 영향을 주지 않음
- 점의 좌표를 조회할 때 이중 탐색을 해야 하는 기존 코드와 달리 점의 좌표가 한 튜플로 묶여 조회를 한번만 할 수 있음
