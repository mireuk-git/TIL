# Codekata
## 추억점수
[link](https://school.programmers.co.kr/learn/courses/30/lessons/176963)
### python

<b>1st try:</b>

```python
def solution(name, yearning, photo):
    result = [0]*len(photo)
    for i in range(len(photo)):
        for j in range(len(photo[i])):
            for k in range(len(name)):
                if photo[i][j]==name[k]:
                    result[i]+=yearning[k]
    return result
```
- O(n^3)
- `name`과 `yearning`을 묶어`dictionary`로 구현하는 방법도 있지만 결과적으로 성능은 O(n^3)으로 동일하다. 다만 실제로는 `dictionary`를 더 많이 쓰는듯. 가독성 때문인가

<b>2nd try:</b>

```python
def solution(name, yearning, photo):
    result = [0]*len(photo)
    d = dict(zip(name,yearning))
    for i in range(len(photo)):
        for j in photo[i]:
            if i in name:
                result[i]+=d[j]
    return result
```
- `dict()`로 가독성 개선

### java
```java
class Solution{
    public int[] solution(String[] name, int[] yearning,String[][] photo){
        int[] result= new int[photo.length];
        for (int i=0;i<photo.length;i++){
            for (String s: photo[i]){
                for (int j=0;j<name.length;j++){
                    if (s.equals(name[j])) result[i]+=yearning[j];
                }
            }
        }
        return result;
    }
}
```

## 바탕화면 정리
[link](https://school.programmers.co.kr/learn/courses/30/lessons/161990)
### python
```python
def solution(wallpaper):
    result=[]

    for i in range(len(wallpaper)):
        for j in range(len(wallpaper[0])):
            if wallpaper[i][j]=='#':
                result.append(i)
                break
        if len(result)==1: break

    for i in range(len(wallpaper[0])):
        for j in range(len(wallpaper)):
            if wallpaper[j][i]=='#':
                result.append(i)
                break
        if len(result)==2: break

    for i in range(len(wallpaper)-1,-1,-1):
        for j in range(len(wallpaper[0])):
            if wallpaper[i][j]=='#':
                result.append(i+1)
                break
        if len(result)==3: break

    for i in range(len(wallpaper[-1])-1,-1,-1):
        for j in range(len(wallpaper)):
            if wallpaper[j][i]=='#':
                result.append(i+1)
                break
        if len(result)==4: break

    return result
```
- `wallpaper`를 위쪽, 왼쪽, 아래쪽, 오른쪽 순서대로 탐색, `#`이 나오는 순간 `result`에 저장하고 탐색 종료
