# Codekata
## 10419
[link](https://www.acmicpc.net/problem/10419)
### python
```python
t=int(input())
l=[0]
for test_case in range(t):
    d=int(input())
    while l[-1]<d: l.append(len(l)*(len(l)+1))
    for i in range(len(l)-1,-1,-1):
        if l[i]<=d:
            print(i)
            break
```
- `l`: `i`분 지각했을 때 전체 수업시간 중 수업을 안듣게 되는 시간 저장, 경계값 역할
- `d`가 `l[i]`보다 작다면 `i`분 지각이 가능함, 이중에서 최대값 검색하기 위해 역순으로 `l` 탐색
- `l`에 저장된 데이터가 `d`를 비교하기에 부족하다면 `d`보다 크거나 같은 값을 찾을 때까지 `l`의 값을 추가함

### java
```java
import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;
import java.util.ArrayList;
import java.util.List;

public class p10419 {
    public static void main(String[] args) throws IOException{
        BufferedReader br = new BufferedReader(new InputStreamReader((System.in)));
        int t = Integer.parseInt(br.readLine());
        List<Integer> l = new ArrayList<>();
        l.add(0);
        for (int testcase=0;testcase<t;testcase++){
            int d = Integer.parseInt(br.readLine());
            while (d>l.get(l.size()-1)) l.add(l.size()*(l.size()+1));
            for (int i=l.size()-1;i>=0;i--){
                if (l.get(i)<=d){
                    System.out.println(i);
                    break;
                }
            }
        }
    }
}
```
