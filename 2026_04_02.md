### python
<b>1st try:</b>

```python
t = int(input())
for testcase in range(t):
    k,n = input().split()
    d=int(n)
    n=n[::-1]
    o=0
    for i in range(len(n)):
        if int(n[i])<8: o+=int(n[i])*8**i
        else: 
            o=0
            break
    h=0
    for i in range(len(n)): h+=int(n[i])*16**i
    print(k, o, d, h)
```
- 8진수로 해석한 `o`를 먼저 처리하고 16진수로 해석한 `h`를 처리
- `o`의 처리 중 자리수 중에 8,9가 등장한다면 즉시 `o`의 처리를 멈추고 0으로 저장한 다음 `h`의 처리로 넘어감
- 각 수의 해석은 각 자리수마다 각각 8과 16의 거듭제곱을 곱해 처리
- 입력받은 `n`은 int로 저장하지 않고 string형으로 저장해 슬라이싱으로 역순서로 변환, 이후 한자리씩 숫자형으로 변환하며 계산 진행

### java
<b>1st try:</b>

```java
import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;
import java.util.StringTokenizer;

public class p13877{
    public static void main(String[] args) throws IOException{
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        int t = Integer.parseInt(br.readLine());
        for (int test=1;test<=t;test++){
            StringTokenizer st = new StringTokenizer(br.readLine());
            st.nextToken();
            int n = Integer.parseInt(st.nextToken());
            int d = n;
            int o = 0, h = 0;
            int i=0;
            while(n>0){
                if (o>=0 && n%10<8) o+=(n%10)*Math.pow(8, i);
                else o=-1;
                h+=(n%10)*Math.pow(16,i);
                n/=10;
                i++;
            }
            if (o<0) o=0;
            System.out.println(test+" "+o+" "+d+" "+h);
        }
    }
}
```
- python과 같은 동작이나 `n`의 입력을 정수형으로 받고 추가적인 연산을 진행해 각 자릿수를 분리, `o`와 `h`의 연산 동시에 진행

<b>2nd try:</b>

```java
public class p13877{
    public static void main(String[] args) throws IOException{
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        int t = Integer.parseInt(br.readLine());
        for (int i=0;i<t;i++){
            StringTokenizer st = new StringTokenizer(br.readLine());
            int k = Integer.parseInt(st.nextToken());
            String s = st.nextToken();

            int oct = 0;
            boolean validOct = true;
            for (char c: s.toCharArray()){
                if (c=='8'||c=='9'){
                    validOct = false;
                    break;
                }
            }
            if (validOct) oct = Integer.parseInt(s,8);
            
            int dec = Integer.parseInt(s,10);
            int hex = Integer.parseInt(s,16);
            System.out.println(k+" "+oct+" "+dec+" "+hex);
        }
    }
}
```
- java도 `Integer.parseInt()`로 각 진법을 구현해 두었다.
