# CodeKata
### python
#### 14723
[link](https://www.acmicpc.net/problem/14723)

```python
n=int(input())
a,b,s=1,1,1
while s<n:
    b+=1
    s+=b
for i in range(s,n,-1):
    a+=1
    b-=1
print(a,b)
```
- 피보나치 수열처럼 분모가 되는 수끼리 더해나감으로써 빠르게 분모에 올 숫자를 구할 수 있음, 거기서부터 목표로 한 n번째 분수까지 세어나감
- 실제로는 분자를 대신 구해 역순으로 `s`번째에서 `n`번째 분수까지 `a`와 `b`를 계산
