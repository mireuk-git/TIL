### python
```python
import sys
input = sys.stdin.readline

t=int(input().strip())
for test_case in range(t):
    a,b = input().strip().split()
    a_digit,b_digit = [],[]
    for i in range(len(a)-len(b)): b_digit.append(1)
    for i in range(len(b)-len(a)): a_digit.append(1)
    a_digit+=map(int,list(a))
    b_digit+=map(int,list(b))
    cal1 = ''
    for i in range(len(a_digit)):
        cal1+=str(a_digit[i]*b_digit[i])
    if int(cal1)==int(a)*int(b): print(1)
    else: print(0)
```
- 문제 알아먹기가 힘들었는데, 문제의 그림대로 단순히 각 자릿수끼리만 계산해 결과를 한줄로 주욱 나열한 것을 답으로 친다. 일의자리는 일의자리대로만 계산하며, 계산결과가 십의자리인 경우에도 그대로 전체 답에 붙인다.
- 두 수 `a`, `b`를 int형이 아닌 string형으로 입력받아 각 자릿수대로 쪼개 list에 저장, 각 자릿수끼리 곱셈을 진행해 `cal1`에 string형태로 concat(이어붙이기)
- 계산도중 자릿수가 짧은 수의 자리가 비어있으면 긴 숫자를 그대로 보존하므로 짧은 쪽에 그만큼 '1'을 추가한다
- 어차피 `a`가 짧든 `b`가 짧든 둘중 하나일테니 `a`가 보정되면 `b`의 보정이 필요없고 `b`가 보정되어야 한다면 `a`가 보정되지 않은 상태다. 따로 보정 전과 후의 `a`를 구별할 필요 없다.

### java
```java
import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;
import java.util.StringTokenizer;

public class p33557 {
    public static void main(String[] args)throws IOException{
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        int t = Integer.parseInt(br.readLine());
        for (int i=0;i<t;i++){ // t번 반복
            // 입력
            StringTokenizer st = new StringTokenizer(br.readLine());
            String aString = st.nextToken();
            long a = Integer.parseInt(aString);
            String bString = st.nextToken();
            long b = Integer.parseInt(bString);

            // 더 짧은 수 보정
            if (aString.length()<bString.length()){
                aString = "1".repeat(bString.length()-aString.length()).concat(aString);
            }
            else{
                bString = "1".repeat(aString.length()-bString.length()).concat(bString);
            }

            // 각 자리마다 곱셈, 문자열로 concat
            String cal = "";
            for (int j=0;j<aString.length();j++){
                cal+=(aString.charAt(j)-'0')*(bString.charAt(j)-'0');
            }
            // 비교 후 출력
            if (Long.parseLong(cal)==a*b) System.out.println(1);
            else System.out.println(0);
        }
    }
}
```
- 각주로 파트 구별
- 보정 부분은 `repeat()`이 음수를 무시하지 못해서 `if`문으로 나누어야 했다
