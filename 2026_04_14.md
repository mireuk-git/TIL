# Codekata
## 11070
[link](https://www.acmicpc.net/problem/11070)
### python
```python
t=int(input())
for _ in range(t):
    n,m = map(int,input().split())
    score=[[0,0] for i in range(n+1)]
    for i in range(m):
        a,b,p,q = map(int,input().split())
        score[a][0] += p
        score[a][1] += q
        score[b][0] += q
        score[b][1] += p
    w=[0]*(n+1)
    for i in range(1,n+1):
        try:
            w[i] = (score[i][0]**2)/(score[i][0]**2+score[i][1]**2)
        except ZeroDivisionError:
            w[i]=0
    w=sorted(w[1:],reverse=True)
    print(int(w[0]*1000))
    print(int(w[-1]*1000))
```
- 모든 팀당 총 득점/실점 수 계산해서 `score`에 저장(`score[i][0]`:득점, `score[i][1]`: 실점)
- 계산된 총 득점/실점 수 바탕으로 각 팀마다 피타고라스 기댓값(`w`) 계산
- `score[0]`,`w[0]`은 존재하지 않는 팀의 데이터이므로 정렬시 제외
- 정렬하여 최댓값, 최솟값 순서대로 출력
