## 4880

### python
```python
import sys
input=sys.stdin.readline

while (True):
    a,b,c = map(int,input().strip().split())
    if (a==0 and b==0 and c==0): break
    if (b-a == c-b): print("AP", 2*c-b)
    elif (b/a == c/b): print("GP", c**2//b)
```
