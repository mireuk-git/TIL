### 9351
[link](https://www.acmicpc.net/problem/9351)
#### python

<b>1st try:</b>

```python
import sys
input=sys.stdin.readline

t = int(input().strip())
for test_case in range(1,t+1):
    palin=[]
    x=0
    s=input().strip()
    for j in range(len(s),0,-1):
        for i in range(j-1):
            substring=s[i:j]
            if substring == substring[::-1]:
                if len(substring)==x:
                    palin.append(substring)
                elif len(substring)>x:
                    palin=[substring]
                    x=len(substring)
    print(f"Case #{test_case}:")
    for i in palin[::-1]:
        print(i)
```
- bruteforce
- `i`와 `j` 사이의 `substring`이 palindrome인지 아닌지 판별하기
- palindrome이 맞다면 `x`보다 큰지, `x`와 같은지 판별
- `x`보다 크다면 `palin` 리스트를 초기화
- `x`와 같다면 `palin`리스트에 추가
- 오른쪽에서부터 추가됐다고 장담할 수 없어서 폐기됨

<b>2nd try:</b>

```python
import sys
input=sys.stdin.readline

t = int(input().strip())
for test_case in range(1,t+1):
    palin=[]
    s=input().strip()
    for x in range(len(s),1,-1):
        for i in range(len(s),x-1,-1):
            substring=s[i-x:i]
            if substring == substring[::-1]:
                palin.append(substring)
        if len(palin)>0: break
    print(f"Case #{test_case}:")
    for i in palin:
        print(i)
```
- 길이`x`를 1차 기준으로 선정, 이후 오른쪽을 기점으로 2차기준 삼아 palindrome 추출
