### 32572
[link](https://www.acmicpc.net/problem/32572)
#### python
```python
import sys
input=sys.stdin.readline

semaphore = {'DownLeft/Down':'A','DownLeft':'B','DownLeft/Top':'C',
             'DownTop':'D','DownTop/Right':'E','DownRight':'F',
             'DownDown/Right':'G','Left/DownLeft':'H','Left/DownLeft/Top':'I',
             'TopRight':'J', 'Left/DownTop':'K','Left/DownTop/Right':'L',
             'Left/DownRight':'M','Left/DownDown/Right':'N','LeftLeft/Top':'O',
             'LeftTop':'P','LeftTop/Right':'Q','LeftRight':'R','LeftDown/Right':'S',
             'Left/TopTop':'T','Left/TopTop/Right':'U','TopDown/Right':'V',
             'Top/RightRight':'W','Top/RightDown/Right':'X','Left/TopRight':'Y','RightDown/Right':'Z'}
semaphorify = {}
for i in semaphore.keys(): semaphorify[semaphore[i]] = i

n,c = map(int,input().strip().split())
s=''
for i in range(n):
    flag=[]
    key=''
    for j in range(9):
        flag.append(input())
    if flag[5][4]=='#': key+='Down'
    if flag[5][3]=='#': key+='Left/Down'
    if flag[4][3]=='#': key+='Left'
    if flag[3][3]=='#': key+='Left/Top'
    if flag[3][4]=='#': key+='Top'
    if flag[3][5]=='#': key+='Top/Right'
    if flag[4][5]=='#': key+='Right'
    if flag[5][5]=='#': key+='Down/Right'
    s+=semaphore[key]

encrypt=''
for i in s:
    encrypt+=chr((ord(i)-65+c)%26+65)
for i in encrypt:
    print(semaphorify[i])

```
- 이모지 만드는게 제일 극혐(결국 때려쳤다. 안해.)
- 하려면 26개 알파벳 전부 다 이모지로 딕셔너리에 저장하는게 편하겠다


# 정보처리
### 디스크 스케줄링
#### C-SCAN
한 방향으로만 이동하며 정렬

>01\. 트랙 번호가 `0부터 199`인 200개의 트랙을 가진 디스크가 있다. 디스크 스케줄링 기법 중 `C-SCAN`을 사용하여 다음과 같은 작업 대기 큐(디스크 큐)의 작업을 수행하는 경우, 처리되는 트랙의 `순서`를 바르게 나열하시오. (단, 현재 디스크 헤드는 `트랙 35에서 트랙47`로 넘어왔다고 가정한다.) 
> 대기큐: 139,22,175,86,13,158

47 > 86 > 139 > 158 > 175 > (199) > (0) > 13 > 22

>02\. 현재 헤드의 위치가 `50`에 있고, 요청 대기열의 순서가 다음과 같을 경우, `C-SCAN` 스케줄링 알고리즘에 의한 헤드의 총 이동거리는 얼마인가? (단, 현재 헤더의 이동 방향은 `안쪽`이며, `안쪽의 위치는 0`으로 가정한다.)
> 대기큐: 100,180,40,120,0,130,70,80,150,200

바깥쪽 끝의 위치에 대해 명시된 바가 없으므로 대기큐 중 가장 바깥쪽인 200을 바깥쪽 끝으로 간주한다.

50 > 40 > 0 > 200 > 180 > 150 > 130 > 120 > 80 > 70
50+200+130 = 380
