#### java
```java
import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;
import java.util.StringTokenizer;

public class p2998 {
    public static void main(String[] args) throws IOException{
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StringTokenizer st = new StringTokenizer(br.readLine());
        String b = st.nextToken();
        b = "0".repeat(3-b.length()%3) + b;
        String o = "";
        for (int i=0;i<b.length();i+=3){
            int tmp = 0;
            for (int j=0;j<3;j++){
                tmp*=2;
                tmp+=Character.getNumericValue(b.charAt(i+j));
            }
            o+=tmp;
        }
        if (o.charAt(0)=='0') System.out.println(o.substring(1));
        else System.out.println(o);
    }
}
```
- `String`에 `int`붙이기: `String`에 `+`연산자로 `int` 붙이기
- `char`에서 `int`로: `Character.getNumericValue(c)`
- Substring: `s.substring(start)`

# 정보처리기사
### 크론표현식
cron - 소프트웨어 예약 실행기, 특정 시간에 작업을 자동으로 실행
크론표현식 - cron의 문자열 형식의 시간 설정 방식
리눅스 기본 크론: 분 시 일 월 요일
쿼츠 등의 라이브러리: 초 분 시 일 월 요일 연도
|필드명|허용범위|허용 특수문자|
|초|00-59|`*`,`,`,`-`,`\`|
|분|00-59|`*`,`,`,`-`,`\`|
|시|00-23|`*`,`,`,`-`,`\`|
|일|01-31|`*`,`,`,`-`,`\`,`L`,`W`|
|월|01-12 / JAN-DEC|`*`,`,`,`-`,`\`|
|요일|0-6 / SUN-SAT|`*`,`,`,`-`,`\`,`L`,`#`|
|연도|1970-2099|`*`,`,`,`-`,`\`|

- `*` - 모든 값
- `-` - 범위
- `,` - 목록 추가
- `/` - 증감 계수


> 01\. 어떠한 작업을 `주기적으로 실행`시키기 위한 `리눅스 명령`과 `데몬`을 쓰시오. 

crontab, crond

crontab - 명령어, crontab -e로 실행
crond - cron에서 돌아가고 있는 데몬

> 02\. 다음 crontab 설정에 대해서 약술하시오.
> */30 * * * * /etc/backup.sh

crontab - 분 시 일 월 요일 프로그램
/etc/backup.sh 위치의 프로그램을 매 30분마다 실행

backup.sh가 30분마다 동작을 한다.

> 03\. `매주` `월요일` `오전 10시`에 `/etc/check.sh`가 실행되도록 설정하는 `cron`을 작성하시오.

10시 정각에만 실행되어야 하기에 '분' 항목에 `*`이 들어가면 안된다.
`0 10 * * MON /etc/check.sh`
`0 10 * * 1 /etc/check.sh`

> 04\. `일요일부터 화요일까지` `오후 4시 30분`에 `/etc/batch.sh`가 실행되도록 설정하는 cron을 작성하시오.

`30 16 * * 0-2 /etc/batch.sh`

>05\. `일요일부터 목요일까지` `오전 2시 30분, 오후 2시 30분`에 `/etc/batch.sh`가 실행되도록 설정하는 cron을 작성하시오.

`30 02,14 * * 0-4 /etc/batch.sh`

> 06\. `1월부터 12월까지` `2개월마다` `1일`, `오전 1시` `10분`에 `/etc/batch.sh`가 실행되도록 설정하는 cron을 작성하시오.

`10 01 1 1-12/2 * /etc/batch.sh`
