# Codekata
### 10407
[link](https://www.acmicpc.net/problem/10407)
#### python
```python
h=int(input())
if h == 1: print(2)
else: print(1)
```
- 수학적 지식이 있다면 쉽게 풀 수 있다(난 없어서 못풀었다)
h=1, 2 mod 3 = 2 = -1 mod 3
h=2, 2^2 mod 3 = 1 = (-1)^2 mod 3
h>=2, ((2^2)^2...)^2 mod 3 = ((-1)^2...)^2 mod 3 = 1
- 즉 h>2 이상이라면 출력값은 항상 1이 되게 된다.
- 2tower값을 일일이 계산하기에는 python의 연산 성능이 그리 좋지 않고 값도 너무 커지므로 문제의 입력 범위에선 당연히 문제가 생기게 되므로, 접근법은 수학적 접근을 이용하는 것 뿐으로 제한되다시피 한다.
- 그냥 수학문제 푸는 것이므로 java코드는 생략한다.
