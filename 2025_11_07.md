# Codekata
### 14809
[link](https://www.acmicpc.net/problem/14809)
#### python
```python
import sys
from math import pi
input=sys.stdin.readline

t=int(input().strip())
for test_case in range(1,t+1):
    n,k = map(int,input().strip().split())
    r,top,side=['']*n,[],[]
    for i in range(n):
        r[i],h=map(int,input().strip().split())
        top.append(r[i]**2)
        side.append(r[i]*2*h)
    pans = sorted(zip(r,top,side),key=lambda x: (-x[2]))
    r,top,side = zip(*pans)

    m=0
    for i in range(n):
        s=top[i]+side[i]
        cnt=1
        for j in range(n):
            if cnt==k: break
            if j!=i and r[j]<=r[i]:
                s+=side[j]
                cnt+=1
        if s>m: m=s
    print(f"Case #{test_case}:", m*pi)
```
- 팬케익의 정보를 입력받음과 동시에, 윗면과 옆면의 크기를 계산해서 `top`과 `side`에 저장
- 정보를 입력받을 때 반지름`r`은 작은 팬케이크 위에 큰 팬케이크가 올 수 없음을 고려해 쌓을 수 있는지를 판정할 때 사용되므로 따로 배열로 저장
- `top`,`side`,`r`을 `zip`으로 묶어 `side`를 기준으로 정렬
- 모든 팬케익을 한번씩 돌아가며 밑바닥에 두고 계산하는 브루트포스 진행
- 밑바닥을 정한 후엔 쌓을 것의 `r`이 밑바닥보다 작은지만 확인하며 그리디 알고리즘(이미 `side`의 크기로 정렬됨)으로 큰 순대로 합산
