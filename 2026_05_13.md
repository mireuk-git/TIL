# Codekata
## 지폐 접기
[link](https://school.programmers.co.kr/learn/courses/30/lessons/340199)
### python
```python
def solution(wallet, bill):
    if wallet[0]>wallet[1]: wallet=wallet[::-1]
    if bill[0]>bill[1]: bill=bill[::-1]
    answer=0

    while wallet[0]<bill[0] or wallet[1]<bill[1]:
        bill[1]//=2
        answer+=1
        if bill[0]>bill[1]: bill=bill[::-1]
    return answer
```
- `bill`과 `wallet` 양쪽 모두 짧은 쪽의 index를 0으로 둔다
- `bill`의 긴 쪽(`bill[1]`)이 반이 될때마다 `answer`에 +1
- `bill`의 양쪽 값이 `wallet`의 양쪽 값보다 작을때까지 반복

### java
```java
class Solution{
    int[] swap(int[] l){
        int tmp=l[0];
        l[0]=l[1];
        l[1]=tmp;
        return l;
    }
    public int solution(int[] wallet, int[] bill){
        if (wallet[0]>wallet[1]) swap(wallet);
        if (bill[0]>bill[1]) swap(bill);
        int answer=0;
        while (wallet[0]<bill[0] || wallet[1]<bill[1]){
            bill[1]/=2;
            answer++;
            if (bill[0]>bill[1]) swap(bill);
        }
        return answer;
    }
}
```
