# Codekata
## 13155
[link](https://www.acmicpc.net/problem/13155)
### python
- 위의 절반을 보고 식별할 수 있는 숫자는 0,1,4,7
- 아래 절반을 보고 식별할 수 있는 숫자는 4,0
- 즉 조건에 만족하는 가짓수는 `(4**n)*(2**n)`
```python
n = int(input())
print(4**n*2**n)
```
### java
<b>1st try:</b>

```java
import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;

public class p13155{
    public static void main(String[] args) throws IOException{
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        int n = Integer.parseInt(br.readLine());
        long i = 8;
        System.out.println(Math.powExact(i, n));
    }
}
```
- `java`의 경우 `int`와 `long`의 범위가 구분되기 때문에 계산하는 변수를 `long`으로 지정해 주어야 한다.

<b>2nd try:</b>

- `Math.powExact(long, int)`가 인식되지 않았다. 
```java
import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;

public class p13155{
    public static void main(String[] args) throws IOException{
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        int n = Integer.parseInt(br.readLine());
        long r = 1;
        for (int i=0;i<n;i++) r*=8;
        System.out.println(r);
    }
}
```
- `powExact()`를 for문을 이용해 직접 구현
