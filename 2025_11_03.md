# CodaKata
### 30189
[link](https://www.acmicpc.net/problem/30189)
#### python
```python
n,m = map(int,input().split())
r=0
f=1
for s in range(min(n,m)+1):
    r+=f
    f+=1
f-=1
for s in range(min(n,m)+1,max(n,m)+1):
    r+=f
f-=1
for s in range(max(n,m)+1,n+m+1):
    r+=f
    f-=1
print(r)
```
- 수식에서 변수 하나가 정해지면 다른 하나의 변수는 값이 고정
- 크게 경우의 수가 세가지로 나뉨
- `0<=i<=min(n,m)`일 경우 가짓수가 1부터 1씩 늘어남
- `min(n,m)+1<=i<=max(n,m)`일 경우 가짓수가 `min(n,m)`으로 고정
- `max(n,m)<=i<=n+m`일 경우 가짓수가 1씩 줄어 최종적으로 1이 됨
