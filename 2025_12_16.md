# Codekata
### 12755
[link](https://www.acmicpc.net/problem/12755)
#### python
```python
n=int(input())
d=1
c=9
start=1
while n > d*c:
    n-=d*c
    d+=1
    c*=10
    start*=10

t=(n-1)//d
s=str(start+t)
print(s[(n-1)%d])
```
- 그냥 숫자열을 문자열로 이어붙이면 시간복잡도가 O(n), n이 최대일 때 시간이 너무 오래 걸림
- 자릿수를 기준삼아 구간을 설정해 구간을 넘어다니며 목표값 탐색
