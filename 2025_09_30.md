#### python
<b>1st try:</b>

```python
import sys
input=sys.stdin.readline
t=int(input().strip())
for _ in range(t):
    a,b,n=map(int,input().strip().split())
    fib=[0,a,b]
    while len(fib)<n+1:
        fib.append(fib[-1]-fib[-2])
    print(fib[n]%1000000007)
```
- 피보나치의 배열버전 풀이
- 시간초과

```python
import sys
input=sys.stdin.readline
t=int(input().strip())
for _ in range(t):
    a,b,n=map(int,input().strip().split())
    l=[a,b,b-a,-a,-b,a-b]
    print(l[(n-1)%6]%1000000007)
```
