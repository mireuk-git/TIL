## 7600
[link](https://www.acmicpc.net/problem/7600)
### python
```python
string=input().lower()
while string != '#':
    d=set()
    for i in string: 
        if ord(i)>=97 and ord(i)<123: d.add(i)
    s=0
    print(len(d))
    string=input().lower()
```
- 입력받은 문자열의 모든 문자를 소문자로 변경
- 각 문자는 존재하는지 확인하되 중복으로 계산되어선 안된다.
- 문자열의 문자가 `a`부터 `z` 사이라면 set `d`에 추가
- `d`의 길이가 곧 중복되지 않는, 각 줄마다 출몰한 알파벳의 개수

### java
```java
import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;
import java.util.HashSet;

public class p7600 {
    public static void main(String[] args) throws IOException{
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        while (true){
            String input = br.readLine().toLowerCase();
            if (input.equals("#")) break;
            HashSet<Character> s = new HashSet<>();
            for (char c : input.toCharArray()){
                if (c>='a' && c<='z') s.add(c);
            }
            System.out.println(s.size());
        }
    }
}
```
- 처음으로 `BufferedReader`만 쓰고 `StringTokenizer`를 쓰지 않았다. `StringTokenizer`는 python의 `split()`같은 개념으로 이해하면 될 거 같다. 
-  `String.toLowerCase()`: 문자열 전체 소문자로 변환
- 조건을 만족하면 HashSet `s`에 저장, `s.size()`가 곧 알파벳 개수
