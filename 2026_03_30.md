# Codekata
## 15161
[link](https://www.acmicpc.net/problem/15161)
### python
```python
k=int(input())
mat=[[k+1 for j in range(10)] for i in range(10)]
for w in range(1,k+1):
    l = list(map(int,input().split()))
    for i in range(3):
        for j in range(10):
            mat[l[i]-1][j]=k+1-w
    for i in range(3):
        for j in range(10):
            mat[j][l[i+3]-1]=k+1-w
for i in mat:
    for j in i:
        print(j,end=" ")
    print()
```
- 한 주가 지날 때마다 `mat`의 모든 요소를 방문하면서 1씩 더하는 건 시간복잡도 상 효율이 매우 안좋다. 처음부터 계산해서 최종값으로 예상되는 값을 반영하는게 더 효율이 좋다.
### java
```java
import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;
import java.util.StringTokenizer;

public class p15161 {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        int k = Integer.parseInt(br.readLine());
        int[][] mat = new int[10][10];
        for (int i=0;i<10;i++){
            for (int j=0;j<10;j++) mat[i][j] = k+1;
        }
        for (int w=1;w<=k;w++){
            StringTokenizer st = new StringTokenizer(br.readLine());
            for (int i=0;i<3;i++){
                int r = Integer.parseInt(st.nextToken());
                for (int j=0;j<10;j++) mat[r-1][j] = k+1-w;
            }
            for (int i=0;i<3;i++){
                int c = Integer.parseInt(st.nextToken());
                for (int j=0;j<10;j++) mat[j][c-1] = k+1-w;
            }
        }
        for (int[] i: mat){
            for (int j: i) System.out.print(j+" ");
            System.out.println();
        }
    }
}
```
