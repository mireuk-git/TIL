# Codekata
### 10041
[link](https://www.acmicpc.net/problem/10041)
#### python
```python
w,h,n=map(int,input().split())
x1,y1=map(int,input().split())
cnt=0
for i in range(n-1):
    flag=True
    x2,y2 = map(int,input().split())
    if x2<x1:
        x1,x2=x2,x1
        y1,y2=y2,y1
        flag=False
    if y2<y1:
        cnt+=abs(x2-x1)+abs(y1-y2)
    else:
        cnt += max(abs(x2-x1),abs(y2-y1))
    if flag: x1,y1=x2,y2
print(cnt)
```
- `x1>x2`로 고정
- `y2<y1`, 즉 우하향 방향의 대각선이라면 맨해튼 거리 사용
- `y2>y1`, 즉 우상향 방향의 대각선이라면 체비쇼프 거리 사용
- `x1<x2`였다면 이미 거리를 합산하기 전에 두 좌표가 바뀌었으므로(=이미 현재 위치를 업데이트 했으므로) `x1>x2`인 경우에 한해서만 거리 계산 후 현재위치 업데이트
