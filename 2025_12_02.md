```python
import sys
input = sys.stdin.readline

t=int(input().strip())
for test_case in range(1,1+t):
    s = input().strip()
    l = len(s)
    i,j = map(int,input().strip().split())
    
    c = s.count('B')*(j//l-(i+l-2)//l)
    i,j = (i-1)%l,(j-1)%l
    if c == 0 and i<j: c = s[i:j+1].count('B')
    else:
        if i != 0: 
            c+=s[i:].count('B')
        if j != l-1: 
            c+=s[:j+1].count('B')
    print(f"Case #{test_case}:",c)
```
- `strip()`을 진행해야 개행/줄바꿈을 무시하고 `l`이 제대로 초기화될 수 있음
- `c`: 'B'의 개수
- `i`와 `j`로 선택된 인덱스 사이에 온전한 `s`가 몇번 들어가는지 계산, `s.count('B')`와 곱해 `c`를 초기화
- 만약 `c`가 0이고 `i`<`j`일 경우, 온전한 `s`의 부분문자열로 판정, 부분문자열의 `count('B')` 진행
- `if`문의 조건이 충족되지 않았을 경우 온전한 `s`외에 다른 부분문자열(같은 마디가 아닌)이 있는 것으로 판정, 남은 부분문자열의 'B'개수 계산
